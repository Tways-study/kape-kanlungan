# Kapé Kanlungan

Single-page marketing site for Kapé Kanlungan, a specialty coffee shop in Sibalom, Antique. Built with [Next.js](https://nextjs.org) (App Router), Tailwind CSS v4, and Framer Motion. See `kape-kanlungan-spec.md` for the full design brief, `PRODUCT.md` for brand/strategy context, and `DESIGN.md` for the visual system.

## Getting started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site. The page auto-updates as you edit files under `app/` and `components/`.

Typefaces are [Fraunces](https://fonts.google.com/specimen/Fraunces) (display) and [Bricolage Grotesque](https://fonts.google.com/specimen/Bricolage+Grotesque) (body/UI), loaded via `next/font/google`.

## Deploy

The easiest way to deploy this project is with the [Vercel Platform](https://vercel.com/new).
